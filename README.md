# Habit Tracker
React 연습용 project입니다.

- add 버튼으로 데이터 추가 할 수 있습니다.
- 각 데이터에 +, -, delete 로 데이터에 점수를 주거나 삭제 할 수 있습니다.
- Reset 버튼으로 데이터 reloading 할 수 있습니다.
